text# Classic Models Sales Dashboard

## Description
An interactive Power BI dashboard analyzing sales data for a model vehicles company from January 2003 to May 2005. Key features include visualizations of net profit by product line, office, customer country, and monthly sales trends with MoM% and YTD calculations. Total sales: $38.57M, net profit: $15.37M.

## Tools Used
- Power BI Desktop for data modeling, DAX calculations, and visualizations.

## Key Insights
- Top product line: Classic Cars ($6.5M net profit).
- Leading offices: France ($6.8M), USA ($5.06M).
- Monthly trends show peak sales in November 2004 ($3.03M).
- Filters for date, product, and drill-downs by country/customer.

## Files
- Sales Overview.pdf: Exported report (2 pages).
- dashboard-page1.png: Screenshot of main dashboard.
- dashboard-page2.png: Screenshot of sales overview and breakdowns.
- classic-models-sales.pbix: Power BI file (open in Power BI Desktop to interact).

## How to View/Run
1. Download the .pbix file.
2. Open in Power BI Desktop (free download from Microsoft).
3. Refresh data if needed and explore filters.

Note: If the .pbix is large, view the PDF/screenshots for a quick overview
